## Data Sources

### Developer Signals
- [Hacker News](https://news.ycombinator.com) - Tech community discussions
- [GitHub Trending](https://github.com/trending) - Rising repositories
- [Lobsters](https://lobste.rs) - Curated tech links
- [Dev.to](https://dev.to) - Developer community

### Startup Signals
- [Product Hunt](https://producthunt.com) - New product launches
- [BetaList](https://betalist.com) - Startup discovery
