# Contributing

Thanks for your interest in contributing!

## Guidelines

1. **One resource per PR** - Makes review easier
2. **Check for duplicates** - Search before adding
3. **Format** - Follow existing table/list format
4. **Quality** - Only add actively maintained resources
5. **No affiliate links** - Keep it clean

## How to Add

1. Fork the repo
2. Add your resource to the appropriate section
3. Submit a PR with a brief description

## Questions?

Open an issue or contact us at [asof.app](https://asof.app)
